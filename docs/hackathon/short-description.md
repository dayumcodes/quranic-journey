# Short Description

**Al-Rihla** (Arabic: *the journey*) is a web app that helps Muslims move beyond Ramadan-season reading into a lasting Quranic habit—through three connected experiences: **Journey** (structured surah exploration with audio and quizzes), **Reflect** (location-aware verse discovery via semantic search), and **Pal** (accountability with a partner via shared goals, progress, and reflections).

Users sign in with Quran Foundation OAuth, engage with authentic verse text, translations, tafsir, and recitation from the Content APIs, and persist personal progress—streaks, goals, collections, and activity—through the User APIs. Al-Rihla is built for clarity, calm design, and real follow-through after the holy month ends.
